# 前言

  • 你无法控制生命中会发生什么，但你可以控制面对这些事情时自己的情绪与行动。
  • 人们要摈弃环境的侵扰，学会追寻生命的意义。

# 自序

  • 你越想成功，就越容易失败。成功就像幸福一样，可遇不可求。它是一种自然而然的产物，是一个人无意识地投身于某一件伟大的事业时产生的衍生品，或者是为他人奉献时的副产品。

# 第一部分 在集中营的经历

  • 爱是人类终身追求的最高目标。
  • 拯救人类要通过爱与被爱。我知道世界上一无所有的人只要有片刻的时间思念爱人，那么他就可以领悟幸福的真谛。
  • 爱一个人可以远远超过爱她的肉体本身。爱在精神和内心方面具有深刻的含义，无论伴侣是否在场，是否健在，爱以什么方式终止是很重要的。
  • 培养幽默感并以一种幽默的态度看待事情，是人在掌握生存艺术时学到的技巧。
  • 人可以克服冷漠，克制暴躁。即使是在可怕的心理和生理条件下，人也能够保持一定的精神自由和意识独立。
  • 有一样东西你是不能从人的手中夺去的，那就是最宝贵的自由，人们一直拥有在任何环境中选择自己的态度和行为方式的自由。
  • 正是在极端困苦的环境下，人才有实现精神升华的机会。
  • 知道为什么而活的人，便能生存。
  • “生命”的意义不是某种含糊的东西，而是非常实在和具体的。它构成人的命运，而每个人的命运都是独特的。你和你的命运无法跟其他任何人及其命运进行比较。生命永不重复，不同问题需要不同的应对。有时你会发现所处的情况需要你采取行动来确定自己的命运，有时你会觉得深思熟虑更为可取，有时你会发现顺其自然是正道。每种情况都有其特殊性，正确的应对也只能有一个。
  • 如果你发现经受磨难是命中注定的，那你就应当把经受磨难作为自己独特的任务。你必须承认，即使在经受磨难时，你也是独特的、孤独的一个人。没有人能够解除你的磨难，替代你的痛苦。你独特的机会就依存于自己承受重负的方式之中。
  • 没能杀死我的，会让我更强壮。

# 存在之本质

  • 人越是忘记自己——投身于某种事业或献身于所爱的人——他就越有人性，越能实现自己的价值。

# 苦难之意义

  • 我们一定不能忘记，即使在看似毫无希望的境地，即使面对无可改变的厄运，人们也能找到生命之意义。
